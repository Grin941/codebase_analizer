Module displays most popular words found in your codebase.

## Requirements

* Python 2.7.X or 3.X

## Installation

```
$ make
```

## Usage

```
$ analize_codebase your_project_path/url_to_clone_repo_from
```

For more information type: ```$ analize_codebase -h```
